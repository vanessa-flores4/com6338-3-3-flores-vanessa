Quiz Assignment!

Instructions

For this assignment, you will need to create a quiz game using JavaScript. The quiz should begin when the user clicks the "Start Quiz" button above. The quiz should ask the user true/false questions and calculate and display the final score at the end of the quiz.

All of your work should be completed within the provided index.js file.

Make a global variable named questionsArr that will contain an array of question objects; this data structure will contain all of the question/answer information for your quiz. Make a function named runQuiz that will contain the code for your quiz (the "Start Quiz" button is already coded to run the runQuiz function). 

    
var questionsArr = []

function runQuiz {}

Inside of your questionsArr array, make an object for each question in your quiz. Your question objects should have two properties:

  1. question - This should be a string containing the question text.
  2. answer - This should be a boolean containing the correct answer.
here is an example of how your questionsArr array should look with a single question object: 

var questionsArr = [
      {
        question: 'JavaScript is the best language',
        answer: true
      }
]

The runQuiz function should then use a for loop to iterate over the questionsArr array and use the question and answer properties of each object in conjunction with the confirm method to ask the user questions and receive and compare answers.

After all questions are asked and answered, the quiz should calculate the percentage correct and then use the alert method to display that percentage to the user.

Tests

Requirements for full credit

To receive full credit for this assignment, your quiz program MUST:

  • Use a function named runQuiz that uses a for loop to iterate over an array of objects stored in a variable named questionsArr.
  • Begin the quiz when the "start quiz" button is clicked.
  • Contain AT LEAST 5 questions.
  • Display the final score as a percentage of the total correctly answered questions.
  • Round final score percentage to a whole number. E.g. "67%" NOT "66.666%"
  • Pass all automated tests.
  • Be deployed to GitHub Pages.
